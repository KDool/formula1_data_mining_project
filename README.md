# formula1_data_mining_project
A Data Mining &amp; Machine Learning Project



This project focuses on applying Machine Learning and Data Mining techniques to historical Formula 1 data in order to extract predictive insights and strategic patterns. In particular, the project aims to develop a three-class classification model that labels each driver race result as podium, points finish, or no points finish, based on factors such as track characteristics, weather conditions, driver statistics, team performance, and previous results.
